# Project Setup Guide

## Download the Repository

Clone the repository to your local machine using the following command:

```sh
git clone https://github.com/hagiu10/Scale_ESP8266
cd Scale_ESP8266
```
## Install arduino-cli

Navigate to the project directory and install arduino-cli:

```sh
run.bat --install-arduino-cli
```

## Configure the Project

Navigate to the project directory and configure the project:

```sh
run.bat --config
```

## Build the Project

Build the project using the following command:

```sh
run.bat --build
```
## Flashing ESP8266

Flashing the memory of ESP8266 with command:

```sh
run.bat --flashing
```
